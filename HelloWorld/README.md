# HelloCMake

This is a minimal C++ project using CMake, designed to build with Visual Studio.

## Build Instructions

1. Open a terminal in this directory.
2. Run the following commands:
   ```
   mkdir build
   cd build
   cmake .. -G "Visual Studio 17 2022"
   cmake --build .
   ```
3. Run the executable from the build directory.

## Requirements
- Visual Studio with C++ and CMake support
- CMake 3.10 or newer
